# java-servlet-gradle

[Demo on Heroku](https://java-servlet-gradle.herokuapp.com/)

### Requirements

  * JDK 16
  * Gradle
  * GNU Make

### Setup

```sh
make setup
```

### Run server

```sh
make start
# Open http://localhost:8080
```
